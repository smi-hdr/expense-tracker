# Expense Tracker

A simple, command-line-based expense tracker application written in Python. This project is designed to help track daily expenses by storing data in a local JSON file.

## Features
- **Add Expense:** Easily log your expenses with a category, description, and amount.
- **View Expenses:** List all recorded expenses.
- **Show Total:** Instantly calculate the sum of all your expenses.
- **Persistent Storage:** Data is saved automatically in `data.json`, so your records remain after closing the program.
